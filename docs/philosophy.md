<h2 style="text-align:center;">9. Differentiation</h2>

<p style="max-width:720px; margin:auto; line-height:1.8; text-align:center;">
The difference is not incremental.

It is architectural.

</p>

<br>

<div style="
  max-width:820px;
  margin:40px auto;
  display:grid;
  grid-template-columns:1fr 1fr;
  gap:20px;
">

  <!-- TRADITIONAL -->
  <div style="
    padding:20px;
    border-radius:14px;
    border:1px solid #e5e7eb;
    background:#ffffff;
  ">
    <h3 style="margin-bottom:10px;">Traditional Systems</h3>

    <ul style="line-height:1.8;">
      <li>Operate in probabilistic generation</li>
      <li>Outputs vary for same input</li>
      <li>Reasoning is implicit or hidden</li>
      <li>No structured decision trace</li>
      <li>Enforcement is external or manual</li>
      <li>Behavior changes at runtime</li>
    </ul>
  </div>

  <!-- MANTHAN -->
  <div style="
    padding:20px;
    border-radius:14px;
    background:#020617;
    color:#ffffff;
    border:1px solid #020617;
  ">
    <h3 style="margin-bottom:10px;">Manthan</h3>

    <ul style="line-height:1.8;">
      <li>Operates in deterministic evaluation</li>
      <li>Same input → same decision</li>
      <li>Reasoning is explicit via contracts</li>
      <li>Full trace for every decision</li>
      <li>Decisions are enforceable by design</li>
      <li>Behavior is fixed and auditable</li>
    </ul>
  </div>

</div>

<br>

<p style="max-width:720px; margin:auto; line-height:1.8; text-align:center;">
This is not a better model.

It is a different layer.

<br><br>

<strong>
From probabilistic systems that generate outputs  
to deterministic systems that make decisions.
</strong>
</p>
